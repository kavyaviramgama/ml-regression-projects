# 🧠 ML Regression Projects

## 🚀 Overview
This repository contains beginner machine learning projects using regression techniques to solve real-world problems.

---

## 📌 Projects

### 🏡 House Price Predictor
Predicts house prices using:
- Median Income
- House Age
- Average Rooms

Models used:
- Linear Regression
- Polynomial Regression
- Random Forest Regression

---

### 📊 Student Performance Predictor
Predicts student marks based on:
- Study Hours
- Sleep Hours
- Attendance

Models used:
- Linear Regression
- Polynomial Regression
- Random Forest Regression

---

## ⚙️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## 📈 Evaluation
- Mean Squared Error (MSE)
- Lower MSE = Better performance

---

## ▶️ How to Run

```bash
pip install pandas numpy scikit-learn matplotlib
```

Run:
```bash
python HousePrice.py
```

or
```bash
python StudentMarks.py
```

---

## 📊 Results
- Random Forest performed best
- Polynomial Regression overfitted on small data
- Linear Regression gave baseline results
